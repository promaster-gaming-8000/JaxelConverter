JaxelConverter Help Guide

Overview

This tool is a graphical interface for vengi-voxconvert, allowing you to convert voxel files without manually typing command-line arguments.

More Information:
GITHUB: https://github.com/vengi-voxel/vengi?tab=readme-ov-file

VENGI-VOXEL: https://vengi-voxel.github.io/vengi/voxconvert/Usage/

Basic Usage

  1. Click Choose File to select your input file
  2. Click Choose Folder to select the output location
  3. Select your desired output format
  4. Adjust settings if needed
  5. Click Convert

Settings Explained

  • Scale (--scale) – Resizes the model
  • Force (--force) – Forces conversion even if issues are detected
  • Surface Only (--surface-only) – Converts only outer surfaces
  • Export Palette (--export-palette) – Saves the model’s color palette
  • JSON (--json) – Outputs model data in JSON format

Supported Formats

  • .vox – MagicaVoxel
  • .vxl – Command & Conquer voxel
  • .obj – Standard 3D model
  • .fbx – Advanced 3D format
  • .gltf – Real-time rendering
  • .stl – 3D printing
  • .ply – Mesh / point cloud

Example Command

vengi-voxconvert.exe --scale 2 --surface-only --input "model.vxl" --output "model.vox"

Notes

  • Input file must be supported
  • Output folder must exist
  • Large models may take time to convert

Advanced Usage

You can manually edit the generated command in the textbox to use custom arguments.
