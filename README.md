JaxelConverter is a desktop application developed in Java using JavaFX that provides a simple and efficient way to convert 3D models between different formats.

The goal of this project is to make model conversion accessible through an intuitive graphical interface, removing the complexity of command-line tools and manual workflows. Users can quickly load model files, configure conversion options, and export them to the desired format.

JaxelConverter is designed to be lightweight, easy to use, and suitable for developers, hobbyists, and creators who frequently work with 3D assets.
